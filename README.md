## Development

### Install tools

```bash
# For details see https://asdf-vm.com/
asdf install
```

### Install dependencies

```bash
$ pnpm i
```

### Type checking

```bash
pnpm check
```

### Build

```bash
pnpm build
```

### Format code

```bash
pnpm format
```

### Run

```bash
pnpm main
```
