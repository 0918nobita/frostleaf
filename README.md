## Development

### Install tools

```bash
asdf plugin-add nodejs
asdf plugin-add pnpm
asdf install
```

### Install dependencies

```bash
$ pnpm i
```

### Type checking

```bash
pnpm check
```

### Build

```bash
pnpm build
```

### Lint

```bash
pnpm lint
```

### Format

```bash
pnpm format
```

### Run Frostleaf CLI

```bash
pnpm main [projectDir]
```
